# MemoLens Frontend Starter

Scaffolded Next.js app-router structure matching the MemoLens plan.

## Run

```bash
cd frontend
cp .env.local.example .env.local
npm install
npm run dev
```

## Included route groups

- `(auth)`: login/register
- `(caregiver)`: dashboard, patients, people, reminders, notes, items, timeline
- `(patient)`: home, query, who

All pages are intentionally minimal placeholders ready for API wiring.
